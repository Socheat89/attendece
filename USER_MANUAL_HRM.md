# សៀវភៅណែនាំប្រើប្រាស់ប្រព័ន្ធគ្រប់គ្រងធនធានមនុស្ស (HRM User Manual)

ឯកសារនេះបង្ហាញពីជំហាននៃការប្រើប្រាស់ប្រព័ន្ធ HRM ចាប់ពីការចូលគណនី រហូតដល់ការបើកប្រាក់ខែ សម្រាប់ទាំង **អ្នកគ្រប់គ្រង (Admin/HR)** និង **បុគ្គលិក (Employees)**។

---

## ផ្នែកទី ១៖ សម្រាប់អ្នកគ្រប់គ្រង (Admin / HR)

ក្រុម HR ឬ Admin គឺជាអ្នករៀបចំប្រព័ន្ធគ្រប់គ្រងទិន្នន័យបុគ្គលិក ការឈប់សម្រាក និងប្រាក់ខែ។

### ១. ការចូលប្រើប្រាស់ (Login & Dashboard)
1. ចូលទៅកាន់តំណភ្ជាប់ (URL) របស់ប្រព័ន្ធ។
2. បញ្ចូល **Email** និង **Password** (គណនី Admin ឬ HR) បន្ទាប់មកចុច Login។
3. នៅលើ Dashboard អ្នកនឹងឃើញទិន្នន័យសង្ខេបដូចជា៖ ចំនួនបុគ្គលិកសរុប អ្នកមកធ្វើការថ្ងៃនេះ អ្នកឈប់សម្រាក និងរបាយការណ៍សង្ខេបផ្សេងៗ។

### ២. ការរៀបចំរចនាសម្ព័ន្ធក្រុមហ៊ុន (Company Setup)
មុននឹងបញ្ចូលបុគ្គលិក អ្នកត្រូវរៀបចំរចនាសម្ព័ន្ធជាមុនសិន៖
1. **សាខា (Branches):** ចូលទៅកាន់ម៉ឺនុយ Branches ដើម្បីបង្កើតទីតាំងការងារ។
2. **ផ្នែក (Departments):** បង្កើតផ្នែកផ្សេងៗក្នុងក្រុមហ៊ុន (ឧទាហរណ៍៖ IT, HR, Marketing)។
3. **តួនាទី (Designations):** បង្កើតឈ្មោះមុខតំណែងសម្រាប់ផ្នែកនីមួយៗ។
4. **ម៉ោងការងារ (Shifts/Schedules):** កំណត់ម៉ោងចូល និងម៉ោងចេញ (ឧទាហរណ៍៖ 8:00 AM - 5:00 PM)។

### ៣. ការបង្កើត និងគ្រប់គ្រងបុគ្គលិក (Employee Management)
1. ចូលទៅកាន់ **Employees** រួចចុចប៊ូតុង **Add New Employee**។
2. បំពេញព័ត៌មានផ្ទាល់ខ្លួន៖ ឈ្មោះ អ៊ីមែល លេខទូរស័ព្ទ លេខកាត។
3. ភ្ជាប់បុគ្គលិកទៅកាន់៖ Branch, Department, Designation, និង Schedule ដែលបានបង្កើត។
4. បង្កើត **គណនី (User Account)** ជូនគាត់ ដើម្បីអាចឱ្យគាត់ Login ចូលប្រព័ន្ធតាមទូរស័ព្ទ។
5. កំណត់ប្រាក់ខែគោល (Basic Salary) របស់គាត់។

### ៤. ការគ្រប់គ្រងវត្តមាន (Attendance Management)
1. **ធ្លាក់ QR Code:** Admin អាច Print QR Code សម្រាប់ឱ្យបុគ្គលិកស្កេន Check-in/Check-out។
2. **តាមដានវត្តមាន (Attendance List):** ពិនិត្យមើលបុគ្គលិកណាដែលមកយឺត (Late) ចេញមុន (Early Leave) ឬអវត្តមាន។
3. **កែប្រែវត្តមាន (Manual Adjustment):** ក្នុងករណីបុគ្គលិកភ្លេចស្កេន HR អាចបញ្ចូលម៉ោងឱ្យគាត់ដោយផ្ទាល់។

### ៥. ការយល់ព្រមលើការឈប់សម្រាក និងថែមម៉ោង (Leave & Overtime)
1. **ច្បាប់ឈប់សម្រាក (Leave Requests):** ចូលទៅម៉ឺនុយ Leave > Requests ដើម្បីមើលការសុំច្បាប់។ Admin អាចចុច **Approve (យល់ព្រម)** ឬ **Reject (បដិសេធ)** ជាមួយមតិយោបល់។
2. **ថែមម៉ោង (Overtime Requests):** ពិនិត្យ និងយល់ព្រមលើការស្នើសុំថែមម៉ោងរបស់បុគ្គលិក ដើម្បីឱ្យប្រព័ន្ធបូកលុយបន្ថែមពេលគណនាប្រាក់ខែ។

### ៦. ការគណនា និងបើកប្រាក់ខែ (Payroll Generation)
ប្រព័ន្ធនឹងគណនាប្រាក់ខែដោយផ្អែកលើ ប្រាក់ខែគោល + ថែមម៉ោង(OT) - ការកាត់ប្រាក់(យឺត/អវត្តមាន)។
1. ចូលទៅកាន់ **Payroll** រួចជ្រើសរើសខែ និងឆ្នាំ។
2. ចុច **Generate Payroll**។ ប្រព័ន្ធនឹងទាញទិន្នន័យវត្តមាន និងច្បាប់មកគណនាសរុប។
3. HR អាចពិនិត្យផ្ទៀងផ្ទាត់ឡើងវិញ មុននឹងចុច **Approve/Lock** ។
4. បន្ទាប់ពី Approve បុគ្គលិកអាចមើលឃើញ វិក្កយបត្រប្រាក់ខែ (Payslip) ក្នុងគណនីរបស់ពួកគេ។

## ផ្នែកទី ២៖ សម្រាប់បុគ្គលិក (Employee)

បុគ្គលិកមានគណនីផ្ទាល់ខ្លួនសម្រាប់ប្រើប្រាស់លើទូរស័ព្ទ ឬកុំព្យូទ័រ ដើម្បីកត់ត្រាវត្តមាន និងសំណូមពរផ្សេងៗដោយខ្លួនឯង។

### ១. ការចូលប្រើប្រាស់ (Login)
1. ទទួលបាន Email និង Password ពីផ្នែក HR។
2. ចូលទៅកាន់ប្រព័ន្ធ (Login) នោះអ្នកនឹងឃើញផ្ទាំង Dashboard សម្រាប់បុគ្គលិក។

### ២. ការកត់ត្រាវត្តមាន (Check-in / Check-out)
1. **បើប្រើ QR Code:** ចុចលើម៉ឺនុយ **Scan** រួចយកកាមេរ៉ាទូរស័ព្ទទៅស្កេន QR Code របស់ក្រុមហ៊ុន។
   - ពេលមកដល់: ប្រព័ន្ធនឹងកត់ត្រាជា IN។
   - ពេលត្រឡប់ទៅវិញ: ស្កេនម្តងទៀត ប្រព័ន្ធនឹងកត់ត្រាជា OUT។
2. *(ជម្រើស)* ប្រព័ន្ធក៏អាចចាប់យកទីតាំង (GPS location) ដើម្បីឱ្យប្រាកដថាអ្នកនៅកន្លែងធ្វើការពេលស្កេន។
3. ប្រព័ន្ធនឹងលោតសារបញ្ជាក់ថាស្កេនបានជោគជ័យ។

### ៣. ការពិនិត្យមើលរបាយការណ៍វត្តមាន (My Attendance)
1. ចូលទៅកាន់ម៉ឺនុយ **My Attendance**។
2. បុគ្គលិកអាចមើលបញ្ជីម៉ោងចូលនិងចេញប្រចាំខែ ដើម្បីដឹងថាខ្លួនឯងមានយឺត ឬអវត្តមានប៉ុន្មានថ្ងៃ។

### ៤. ការស្នើសុំច្បាប់ឈប់សម្រាក (Request Leave)
1. ចូលទៅកាន់ **Leave** > ប៊ូតុង **Request New Leave**។
2. រើសប្រភេទច្បាប់ (ឧទាហរណ៍៖ ឈប់ឈឺ, ឈប់កិច្ចការផ្ទាល់ខ្លួន)។
3. ជ្រើសរើស កាលបរិច្ឆេទចាប់ផ្តើម និងបញ្ចប់។
4. បញ្ចូលមូលហេតុ រួចចុច **Submit**។
5. រង់ចាំ HR យល់ព្រម (អ្នកនឹងឃើញ Status របស់វានៅលើប្រព័ន្ធ)។

### ៥. ការស្នើសុំថែមម៉ោង (Request Overtime)
1. ចូលទៅកាន់ **Overtime (OT)** > ប៊ូតុង **Request OT**។
2. ជ្រើសរើសថ្ងៃខែដែលអ្នកបានធ្វើការថែមម៉ោង រួមទាំងចំនួនម៉ោង (ឧទាហរណ៍៖ ២ ម៉ោង)។
3. បញ្ចូលមូលហេតុនៃការថែមម៉ោង រួចចុច **Submit**។

### ៦. ការមើល និងទាញយកប្រាក់ខែ (View & Download Payslip)
1. ជារៀងរាល់ចុងខែ (ពេល HR បើកប្រាក់ខែរួច) ចូលទៅកាន់ម៉ឺនុយ **Salary** ឬ **Payslip**។
2. អ្នកនឹងឃើញបញ្ជីប្រាក់ខែរបស់ខែនីមួយៗ។
3. ចុច **View** ឬ **Download (PDF)** ដើម្បីមើលលម្អិតអំពីប្រាក់កម្រៃ ប្រាក់បន្ថែម ប្រាក់ដក និងប្រាក់សរុបដែលអ្នកទទួលបាននៅក្នុងខែនោះ។
